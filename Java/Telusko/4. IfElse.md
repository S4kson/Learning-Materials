# Условные операторы в Java (if, else, else-if, тернарный)


## Оператор `if`

* Выполняет блок кода, если условие истинно (`true`).
* Условие всегда должно возвращать `boolean` (в отличие от C/C++).
* Если блок состоит из одной строки, фигурные скобки можно опускать (не рекомендуется).

### Синтаксис
```
if (условие) {
    // код выполняется, если условие == true
}
```

---

## Оператор `if-else`

* Добавляет альтернативный путь выполнения.
* Если условие `true` - выполняется `if`-блок, иначе - `else`-блок.

### Синтаксис
```
if (условие) {
    // true
} else {
    // false
}
```

---

## Оператор `else-if` (лестница)

* Используется для проверки нескольких условий последовательно.
* Как только одно условие становится `true`, его блок выполняется, остальные пропускаются.
* Блок `else` в конце - опционально (выполняется, если ни одно условие не `true`).

### Синтаксис
```
if (условие1) {
    // условие1 == true
} else if (условие2) {
    // условие2 == true
} else if (условие3) {
    // условие3 == true
} else {
    // все условия false
}
```

---

## Тернарный оператор (`? :`)

* Сокращённая форма `if-else` для простых присваиваний.
* Возвращает одно из двух значений в зависимости от условия.

### Синтаксис
```
переменная = (условие) ? значение1 : значение2;
```

### Пример
```
int max = (a > b) ? a : b;   // max = a, если a > b, иначе max = b
```

---

## Вложенные `if`

* `if` или `else` могут содержать внутри другие `if-else`.
* **Dangling else**: `else` всегда привязывается к ближайшему `if` без `else`.

---




## Основные команды

* `if (x > 0) { ... }` - проверка условия.
* `if (x > 0) { ... } else { ... }` - ветвление.
* `if (x > 0) { ... } else if (x < 0) { ... } else { ... }` - множественные условия.
* `int y = (x > 0) ? 1 : -1;` - тернарный оператор.
