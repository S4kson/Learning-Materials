# Как работает Java

## Процесс выполнения Java-кода

### Основные этапы
* **Исходный код** - код, написанный программистом на языке Java (файлы с расширением `.java`).
* **Компиляция** - исходный код преобразуется в байт-код с помощью компилятора `javac`.
* **Байт-код** - промежуточное представление программы (файлы с расширением `.class`). JVM понимает только байт-код, но не исходный Java-код.
* **Исполнение** - байт-код передаётся в JVM (Java Virtual Machine) для дальнейшего выполнения.

### Схема выполнения
`Программист → Java-код → Компилятор (javac) → Байт-код (.class) → JVM → Выполнение`

## Платформенная независимость

* Java является **платформенно-независимым** языком - программу можно запустить на любой машине независимо от аппаратного обеспечения или операционной системы.
* Единственное требование для запуска Java-кода - наличие **JVM** на целевой машине.
* **JVM является платформенно-зависимой** - она должна быть реализована отдельно для каждой операционной системы.

## Точка входа в программу - метод `main`

* Выполнение Java-программы всегда начинается с метода `main`.
* JVM запускает выполнение только с одного первого файла. Если требуется запустить несколько файлов, необходимо указать, какой из них является точкой входа.
* Первый файл **обязательно** должен содержать метод `main` с правильной сигнатурой.

### Сигнатура метода `main`
`public static void main(String args[])`

## Объектно-ориентированность

* Java является **объектно-ориентированным** языком программирования.
* Объектно-ориентированный подход означает, что всё должно быть объектом.
* Для создания объекта всегда требуется класс.

## Важные моменты

* Для запуска Java-кода требуется JVM, которая исполняет байт-код, а не исходный код.
* Благодаря байт-коду Java-программы могут работать на любых платформах, где установлена подходящая JVM.
* Метод `main` с точной сигнатурой `public static void main(String args[])` является обязательной точкой входа.
* Java поддерживает объектно-ориентированное программирование, что требует использования классов для создания объектов.

## Основные компоненты

* **JDK (Java Development Kit)** - набор инструментов для разработки на Java, включает компилятор `javac` и JRE.
* **JRE (Java Runtime Environment)** - среда выполнения Java, включает JVM и библиотеки классов.
* **JVM (Java Virtual Machine)** - виртуальная машина, исполняющая байт-код.
* **javac** - компилятор Java, преобразует `.java` в `.class` (байт-код).
