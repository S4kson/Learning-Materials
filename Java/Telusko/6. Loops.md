## Цикл `while`

* Выполняет блок кода, пока условие истинно.
* Проверка условия выполняется **перед** каждой итерацией.
* Если условие изначально `false` - тело цикла не выполнится ни разу.

### Синтаксис
```java
while (условие) {
    // тело цикла
}
```

### Пример
```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

---

## Цикл `do-while`

* Выполняет блок кода, затем проверяет условие.
* Тело цикла выполняется **как минимум один раз** независимо от условия.
* Используется для задач, где требуется гарантированное однократное выполнение (например, валидация ввода, меню).

### Синтаксис
```java
do {
    // тело цикла
} while (условие);
```

### Пример
```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

## Цикл `for`

* Компактная структура: инициализация, условие, инкремент - всё в одной строке.
* Идеален для циклов с известным количеством итераций (перебор диапазонов, работа с массивами).

### Синтаксис
```java
for (инициализация; условие; инкремент) {
    // тело цикла
}
```

### Пример
```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

---

## Цикл `for-each` (enhanced for)

* Упрощённая версия `for` для перебора массивов и коллекций.
* Не требует ручного управления индексом.
* Доступен только для чтения (изменение элемента не влияет на исходный массив).

### Синтаксис
```java
for (тип переменная : массив_или_коллекция) {
    // тело цикла
}
```

### Пример
```java
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}
```

## Управление потоком выполнения

### `break`
* Немедленный выход из цикла.
* Работает во всех типах циклов.

```java
for (int i = 0; i < 10; i++) {
    if (i == 5) break;  // выход при i == 5
    System.out.println(i);
}
```

### `continue`
* Пропуск текущей итерации - переход к следующей.
* Работает во всех типах циклов.

```java
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;  // пропуск чётных чисел
    System.out.println(i);
}
```

### Вложенные циклы
* Циклы могут быть вложены друг в друга.
* `break` и `continue` по умолчанию применяются к ближайшему циклу.
* Для выхода из внешнего цикла используется **меченая метка** (labeled break).

```java
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) break outer;
        System.out.println(i + " " + j);
    }
}
```

## Выбор подходящего цикла

| Тип цикла | Когда использовать |
|-----------|-------------------|
| `while` | Количество итераций неизвестно заранее |
| `do-while` | Тело должно выполниться хотя бы один раз |
| `for` | Количество итераций известно заранее |
| `for-each` | Перебор всех элементов массива или коллекции |
