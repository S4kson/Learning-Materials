# Арифметические, реляционные и логические операторы в Java


## Арифметические операторы

| Оператор | Назначение | Пример |
|----------|------------|--------|
| `+` | Сложение | `a + b` |
| `-` | Вычитание | `a - b` |
| `*` | Умножение | `a * b` |
| `/` | Деление | `a / b` |
| `%` | Остаток от деления | `a % b` |

---

## Реляционные операторы

| Оператор | Назначение | Пример |
|----------|------------|--------|
| `==` | Равно | `a == b` |
| `!=` | Не равно | `a != b` |
| `>` | Больше | `a > b` |
| `<` | Меньше | `a < b` |
| `>=` | Больше или равно | `a >= b` |
| `<=` | Меньше или равно | `a <= b` |

* Результат - `boolean` (`true`/`false`).

---

## Логические операторы

| Оператор | Назначение | Пример |
|----------|------------|--------|
| `&&` | Логическое И (short-circuit) | `a && b` |
| `\|\|` | Логическое ИЛИ (short-circuit) | `a \|\| b` |
| `!` | Логическое НЕ | `!a` |
---

## Составные операторы присваивания

| Оператор | Эквивалент |
|----------|------------|
| `+=` | `a = a + b` |
| `-=` | `a = a - b` |
| `*=` | `a = a * b` |
| `/=` | `a = a / b` |
| `%=` | `a = a % b` |

* Поведение идентично C#.

---

## Инкремент и декремент

| Оператор | Назначение |
|----------|------------|
| `++` | Увеличение на 1 |
| `--` | Уменьшение на 1 |

* **Префиксная форма** (`++a`) - сначала изменение, потом использование.
* **Постфиксная форма** (`a++`) - сначала использование, потом изменение.
* Поведение идентично C#.

---

## Приоритет операторов (сверху вниз)
