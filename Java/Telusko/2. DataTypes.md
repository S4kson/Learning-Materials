# Переменные в Java

## Определение
* **Переменная** - именованная область памяти для хранения данных.
* Имеет **тип**, **имя** и **значение**.

## Объявление и инициализация
* Объявление: `тип имя;` (например, `int number;`)
* Инициализация: `тип имя = значение;` (например, `int number = 10;`)
* Изменение значения: `имя = новое_значение;`

## Правила именования
* Начинается с буквы, `$` или `_`.
* Далее буквы, цифры, `$`, `_`.
* Регистрозависимо.
* Нельзя использовать ключевые слова (class, int, public и т.д.).
* Рекомендуется **camelCase** (например, `myVariable`).

## Виды переменных

| Вид | Место объявления | Время жизни |
|-----|------------------|-------------|
| **Локальные** | Внутри метода | Существуют только во время выполнения метода. **Требуют явной инициализации** перед использованием. |
| **Переменные экземпляра** | Внутри класса, вне методов | Существуют для каждого объекта. Получают значения по умолчанию (0, false, null). |
| **Статические** | С ключевым словом `static` | Существуют на уровне класса (общие для всех объектов). Получают значения по умолчанию. |

## Важные моменты
* Java - строго типизированный язык: тип фиксируется при объявлении.
* Локальные переменные **обязательно** инициализируются до использования (ошибка компиляции).
* Переменные экземпляра и статические имеют значения по умолчанию.

# Типы данных в Java

## Две категории
* **Примитивные (Primitive)** - хранят значения непосредственно (8 типов).
* **Ссылочные (Reference)** - хранят ссылки на объекты (классы, интерфейсы, массивы).

## Примитивные типы

### Целочисленные

| Тип | Размер | Диапазон |
|-----|--------|----------|
| `byte` | 1 байт | -128 … 127 |
| `short` | 2 байта | -32 768 … 32 767 |
| `int` | 4 байта | -2³¹ … 2³¹-1 |
| `long` | 8 байт | -2⁶³ … 2⁶³-1 |

### С плавающей точкой

| Тип | Размер | Диапазон | Точность |
|-----|--------|----------|----------|
| `float` | 4 байта | ~ ±3.4×10³⁸ | ~7 знаков |
| `double` | 8 байт | ~ ±1.8×10³⁰⁸ | ~15 знаков |

### Символьный и логический

| Тип | Размер | Описание |
|-----|--------|----------|
| `char` | 2 байта | Один символ (Unicode) |
| `boolean` | ~1 байт (логически) | `true` или `false` |

## Ссылочные типы
* Хранят **адрес объекта** в куче (heap), а не само значение.
* Значение по умолчанию - `null`.
* Примеры: `String`, массивы (`int[]`), пользовательские классы, интерфейсы.

## Приведение типов
* **Неявное (автоматическое)** - от меньшего к большему (без потерь): `int → long → double`.
* **Явное (ручное)** - от большего к меньшему (с возможной потерей данных): `(тип) значение`.

### Пример
```
double d = 9.99;
int i = (int) d;  // i = 9 (дробная часть отброшена)
```

## Классы-оболочки (Wrapper)
* Для каждого примитива есть обёртка: `Integer` для `int`, `Double` для `double` и т.д.
* Позволяют использовать примитивы в коллекциях и как объекты.
* Автоупаковка/распаковка выполняются автоматически.

## Важные моменты
* Примитивы хранят значения, ссылочные - адреса объектов.
* Строки (`String`) - ссылочный тип, но неизменяемы (immutable).
* Размер `boolean` не специфицирован точно (зависит от JVM).

## Основные команды (шпаргалка)
* `int x = 5;` - примитив.
* `String s = "Hello";` - ссылочный тип.
* `Integer obj = 10;` - автоупаковка.
* `int y = obj;` - автораспаковка.
* `double d = x;` - неявное приведение.
* `byte b = (byte) x;` - явное приведение (если x > 127 - потеря).
