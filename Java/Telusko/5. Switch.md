# Оператор `switch` в Java

## Классический `switch` (до Java 11)

### Синтаксис
```java
switch (выражение) {
    case значение1:
        // код
        break;
    case значение2:
        // код
        break;
    // ...
    default:
        // код
}
```

* Выражение вычисляется однократно и сравнивается с метками `case`.
* При совпадении выполняется соответствующий блок кода.
* Отсутствие `break` вызывает **проваливание** (fall‑through) - выполнение продолжается в следующем `case`.
* `default` - исполняется, если ни один `case` не совпал (опционально, но рекомендуется всегда добавлять).

### Допустимые типы для выражения
* `byte`, `short`, `char`, `int`
* `enum`
* `String` (начиная с Java 7)
* `long`, `float`, `double`, `boolean` - **не разрешены** (ошибка компиляции).

## Новый синтаксис `switch` (Java 12+)

### Стрелочный оператор `->`

* Представлен как preview в Java 12, стандартизирован в Java 14.
* **Отключает проваливание** - `break` не требуется, выполняется только один блок.
* Несколько значений перечисляются через запятую.

**Пример:**
```java
switch (day) {
    case MON, FRI -> System.out.println("Рабочий день");
    case SAT, SUN -> System.out.println("Выходной");
    default -> System.out.println("Неизвестный день");
}
```

### `switch` как выражение (возвращает значение)

* `switch` может использоваться в правой части присваивания.
* Возвращаемое значение определяется либо через `->` (если однострочное выражение), либо через блок с `yield`.

**Вариант с `->` (короткий):**
```java
int num = switch (day) {
    case MON, FRI -> 1;
    case WED -> 3;
    default -> 0;
};
```

**Вариант с блоком и `yield` (для нескольких операторов):**
```java
int num = switch (day) {
    case MON, FRI -> {
        System.out.println("Понедельник или пятница");
        yield 1;
    }
    default -> 0;
};
```
* `yield` возвращает значение из блока (аналог `return` для `switch`-выражений).
* `yield` доступен только в выражениях, не в операторах.

---

## Ключевые отличия нового синтаксиса от старого

| Старый (с `:`) | Новый (с `->`) |
|----------------|----------------|
| Проваливание разрешено (требуется `break`) | Проваливание запрещено |
| Только оператор (не возвращает значение) | Может быть выражением (возвращает значение) |
| Группировка значений через несколько `case` | Группировка через запятую в одном `case` |
| Для возврата значений не используется | Используется `yield` в блоках для возврата |

## Основные конструкции

- Классический оператор:  
  ```java
  switch (x) {
      case 1: System.out.println("Один"); break;
      default: System.out.println("Другое");
  }
  ```

- Стрелочный оператор:  
  ```java
  switch (x) {
      case 1, 2 -> System.out.println("Один или два");
      default -> System.out.println("Другое");
  }
  ```

- Switch-выражение с `->`:  
  ```java
  int result = switch (x) {
      case 1 -> 10;
      case 2 -> 20;
      default -> 0;
  };
  ```

- Switch-выражение с блоком и `yield`:  
  ```java
  int result = switch (x) {
      case 1 -> {
          int temp = x * 2;
          yield temp;
      }
      default -> 0;
  };
  ```
