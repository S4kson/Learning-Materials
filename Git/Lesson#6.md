# Справка в Git и команда статуса

## Система справки Git

### Базовые команды справки
* **Основная справка**: `git help` - показывает список часто используемых команд
* **Полный список команд**: 
  * `git help -a`
  * `git help --all`

### Типы команд Git
* **Команды интерфейсы** (high-level):
  * Основные, часто используемые команды
  * Примеры: add, rm, commit

* **Команды утилиты** (low-level):
  * Низкоуровневые команды
  * Используются редко

## Получение справки по конкретной команде
* Синтаксис: `git help <команда>`
* Примеры:
  * `git help init`
  * `git help add`
  * `git help commit`

## Инициализация репозитория
* **Создание репозитория**: `git init`
* Создает пустой репозиторий Git в текущей папке
* Автоматически создает основную ветку (master)

## Проверка статуса проекта
* **Команда статуса**: `git status`
* Показывает:
  * Текущую ветку (по умолчанию master)
  * Статус коммитов
  * Файлы для отслеживания
  * Общее состояние проекта
