# Настройка конфигурации Git

## Инициализация репозитория

Для инициализации репозитория необходимо выполнить команду:

`git init`

После выполнения команды создаётся:
* скрытая папка .git;
* внутри папки .git содержатся:
    * папки hooks, info, objects, refs;
    * файл конфигурации config.

## Настройка пользователя

### Локальная конфигурация (для конкретного проекта)

Для установки данных пользователя на уровне проекта требуется выполнить команды:

`git config --local user.email "example@gmail.com"`  
`git config --local user.name "Username"`

### Глобальная конфигурация

Для установки данных пользователя на уровне всей системы требуется выполнить команды:

`git config --global user.email "saksonov.kostya@gmail.com"`  
`git config --global user.name "S4kson"`

## Управление настройками

### Просмотр текущих настроек

Для получения значений отдельных параметров или полного списка настроек необходимо выполнить команды:

`git config user.email`  
`git config user.name`  
`git config --list`

### Удаление настроек

Для удаления конкретного параметра либо целого раздела настроек требуется выполнить команды:

`git config --unset user.email`  
`git config --remove-section user`

## Важные моменты

* Приоритет настроек: локальные настройки (уровень проекта) имеют приоритет над глобальными. При отсутствии локальных настроек применяются глобальные.
* Сохранение изменений: изменения применяются сразу после выполнения команды. Настройки хранятся в файле конфигурации.
